# Spotify-FrontEnd-API
Displaying data with spotify web api.
#Technologies
React and NodeJs
# External Library:
React Router
# How to Run
Open Command Prompt:
1. First install react router using: "npm install --save react-router".
2. Then "npm start".

# Time Spent
6-7 hours (Not in one sitting)
